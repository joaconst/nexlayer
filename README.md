# NexLayer - Professional Agency Website

A production-ready, bilingual (EN/ES) agency website built with Next.js 14, Tailwind CSS, Framer Motion, and TypeScript.

## 🚀 Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript 5
- **Styling:** Tailwind CSS 3.4
- **Animations:** Framer Motion 11
- **Forms:** React Hook Form + Zod
- **Email:** EmailJS
- **Icons:** Lucide React
- **Deployment:** Vercel

## 📋 Prerequisites

- Node.js 18.17 or later
- npm or yarn or pnpm
